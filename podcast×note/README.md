# ポッドキャスト × note 記事管理

このディレクトリは「Talk Beans Talk」ポッドキャストの音声ファイルと、それに基づく note 記事を管理します。

## ディレクトリ構成

```
podcast×note/
├── Podcast/           # 音声ファイル保管庫（シーズン7）
│   └── S7E1_音声.m4a
├── S7E1_note下書き.md
├── S7E1_note完成版.md
└── README.md
```

## 命名規則

- **音声ファイル**: `S7E{番号}_音声.m4a`
- **下書き版**: `S7E{番号}_note下書き.md`
- **完成版**: `S7E{番号}_note完成版.md`

## note 記事の方針

- 話し言葉を読みやすい文語に整える（意味・内容は追加しない）
- フィラー語（えー、あのー）は除去
- Jackの語り口・一人称を保持
- 読者に語りかけるスタイルで自然な日本語に
